
# Customer Churn Prediction (Telecom)

## Objective
Predict customer churn using classic machine learning models and explain predictions using SHAP.

## Models Used
- Logistic Regression
- Random Forest
- XGBoost (Best)

## Techniques
- EDA & Feature Engineering
- Hyperparameter Tuning
- Model Comparison
- SHAP Interpretability

## Results
XGBoost achieved the highest ROC-AUC score and was selected as the final model.

## Business Impact
Actionable churn reduction strategies based on model insights.
